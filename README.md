# simple_shell project 

This simple shell project contains a bash like c shell

## Used compiler

```
gcc -Wall -Werror -Wextra -pedantic -std=gnu89 *.c -o shell
```

## Demonestration


```
./shell
```
##
```
& >> ls

```
